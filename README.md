# HR-ANALYTICS-DASHBOARD
Developed an interactive HR Analytics Dashboard using Power BI to analyze employee data and workforce trends. The dashboard includes KPIs such as Total Employees, Active Employees, Attrition Rate, Average Age, and Experience, along with department-wise, salary-wise, and age-group analysis. 
# HR Analytics Dashboard 📊

## Overview
This project is an interactive **HR Analytics Dashboard** built using **Power BI**. It provides valuable insights into employee data, workforce performance, and attrition trends, helping organizations make data-driven HR decisions.

## Objectives
- Analyze employee attrition.
- Monitor workforce distribution.
- Identify department-wise employee count.
- Compare attrition across salary groups.
- Understand age and experience distribution.
- Provide interactive business insights.

## Dashboard Features
- Total Employees
- Active Employees
- Total Attrition
- Attrition Rate (%)
- Average Age
- Average Experience
- Department-wise Attrition Analysis
- Salary-wise Attrition Analysis
- Age Group Distribution
- Attrition Trend by Experience
- Department-wise Employee Count
- Interactive Department Filter (Slicer)

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
- Python pandas

## Key Insights
- Monitors overall employee strength and active workforce.
- Highlights departments with the highest attrition.
- Analyzes employee attrition based on salary ranges.
- Shows workforce distribution by age and experience.
- Enables HR teams to make informed business decisions.
